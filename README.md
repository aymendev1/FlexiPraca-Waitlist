## How to install

1. Clone this repositoy:

```bash
git clone https://github.com/aymendev1/FlexiPraca-Waitlist.git
```

2. Go into project

```bash
cd FlexiPraca-Waitlist
```

3. Install dependencies

```bash
npm install
```

4. Run project

```bash
npm run dev
```
